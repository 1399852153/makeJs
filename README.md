For leaning how to do a compiler.

```js
let a;
let b;
let c;
let www;
a = 1+7*6;
b = 8*3*2;
www = 0;

while(www<=20){
    www = www + 1;
}

if(a > b){
    c = a;
}else {
    c = b;
}

```
